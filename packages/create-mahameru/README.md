# MahameruJS Project Initializer

[![npm version](https://img.shields.io/npm/v/mahameru.svg)](https://www.npmjs.com/package/mahameru)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> Spin up a new **MahameruJS** project in one command.

`create-mahameru` is the official CLI initializer for MahameruJS. It pulls the starter template, sets your project name, and installs dependencies automatically.

## Quick Start

```bash
npm create mahameru
```

or

```bash
npx create-mahameru
```

and follow the instructions.

## Requirements

- Node.js
- npm
- Internet connection
